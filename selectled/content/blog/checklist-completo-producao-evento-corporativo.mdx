Produzir um evento corporativo envolve dezenas de variáveis simultâneas. Este checklist foi criado a partir de mais de 5.000 eventos realizados pela Select LED e cobre desde o briefing inicial até o relatório pós-evento.

## Fase 1 — Briefing e planejamento (60-90 dias antes)

### Definição estratégica
- [ ] Definir objetivo principal do evento (convenção, lançamento, treinamento, premiação)
- [ ] Confirmar data e hora (incluindo montagem e desmontagem)
- [ ] Estimativa de público (mínimo e máximo)
- [ ] Orçamento total aprovado com margem de contingência (15-20%)
- [ ] Confirmar o CNPJ que emitirá a nota fiscal

### Espaço e logística
- [ ] Confirmar local com contrato assinado
- [ ] Visita técnica ao espaço (elétrica, estrutura, acústica)
- [ ] Verificar carga elétrica disponível (painel de 20m² consome ~4 kW)
- [ ] Confirmar height clearance para painéis suspensos
- [ ] Definir layout completo do palco e plateia

---

## Fase 2 — Contratação de fornecedores (30-45 dias antes)

### Painel de LED
- [ ] Definir tamanho e pitch do painel (use a [calculadora da Select LED](/sao-paulo/paineis-led))
- [ ] Confirmar tipo de uso: indoor ou outdoor/híbrido
- [ ] Confirmar se precisa de redundância de processador
- [ ] Definir se haverá IMAG (câmeras ao vivo) → obrigatório P2.5 ou menor
- [ ] Solicitar planta técnica do painel (peso, pontos de carga)

### Áudio
- [ ] Line array ou caixas? (depende do tamanho do espaço e tipo de evento)
- [ ] Confirmar microfones (bastão, lapela, headset)
- [ ] Mesa de som com operador incluído?
- [ ] Testagem de acústica no local (reverb, eco)

### Iluminação
- [ ] Moving heads para palco e plateia
- [ ] Iluminação de plateia (wash) para filmagem
- [ ] Iluminação de fundo / backdrop
- [ ] Fog ou hazer para efeitos especiais?

### Vídeo e streaming
- [ ] Câmeras para IMAG e/ou transmissão ao vivo
- [ ] Plataforma de streaming (YouTube Live, Zoom, Teams)
- [ ] Internet dedicada (verificar disponibilidade no espaço)
- [ ] Gravação para entrega pós-evento

### Outros
- [ ] Credenciamento (totens, pulseiras, crachás)
- [ ] Contratação de produção executiva
- [ ] Segurança e recepcionistas

---

## Fase 3 — Preparação de conteúdo (15-30 dias antes)

### Material visual para o painel
- [ ] Template master em 1920×1080 (Full HD) ou 3840×2160 (4K)
- [ ] Apresentações padronizadas em 16:9
- [ ] Vídeos de abertura e encerramento (MP4 H.264, áudio embutido)
- [ ] Motion graphics e animações (.mov com fundo transparente se necessário)
- [ ] Playlist de fundo para antes e depois do evento
- [ ] Vinheta de chamada para o palco

### Identidade visual
- [ ] Logotipos dos patrocinadores em vetor ou PNG sem fundo
- [ ] Paleta de cores aprovada pelo cliente
- [ ] Fontes licenciadas para uso comercial

---

## Fase 4 — Montagem (1-2 dias antes)

- [ ] Acesso ao espaço confirmado com horário de início
- [ ] Carga elétrica dedicada marcada com o espaço
- [ ] Piso protegido para passagem de cabos (evitar acidentes)
- [ ] Teste de todos os equipamentos com sinal real
- [ ] Teste de câmeras (foco, enquadramento, IMAG funcionando)
- [ ] Teste de internet e streaming (bandwidth mínimo: 10 Mbps upload por câmera)
- [ ] Rundown completo com operador de painel, áudio e luz
- [ ] Reunião de alinhamento com todos os técnicos

---

## Fase 5 — Dia do evento

### Manhã / pré-evento
- [ ] Check final de todos os equipamentos
- [ ] Teste de som com o microfone que será usado
- [ ] Confirmar plano B para cada equipamento crítico
- [ ] Verificar nobreak e grupos geradores (se externos)

### Durante o evento
- [ ] Operador de painel dedicado (não deixar em automático para eventos complexos)
- [ ] Fotógrafo/câmera posicionado
- [ ] Técnico de áudio na mesa durante todo o evento
- [ ] Ponto de suporte da Select LED disponível via WhatsApp

### Pós-evento (no dia)
- [ ] Backup dos arquivos de gravação
- [ ] Desmontagem organizada e verificação de equipamentos
- [ ] Checklist de devolução do espaço

---

## Fase 6 — Pós-evento (1-7 dias depois)

- [ ] Envio de gravação ao cliente
- [ ] Nota fiscal emitida
- [ ] Pesquisa de satisfação com os participantes
- [ ] Relatório de audiência (se streaming)
- [ ] Reunião de debriefing com a equipe produtora
- [ ] Arquivo do projeto para referências futuras

---

## Bônus: erros mais comuns que evitam com este checklist

1. **Esquecer da carga elétrica** — um painel de 20m² consome tanto quanto 2 ar-condicionados de 12.000 BTUs. Sem circuito dedicado, há risco de queda de energia.
2. **Deixar o conteúdo para o último momento** — mudanças de última hora no material são a principal causa de atraso nos eventos.
3. **Não confirmar o acesso de montagem** — muitos espaços exigem credenciamento antecipado para equipes técnicas.
4. **Ignorar o teste de IMAG** — a latência entre câmera e tela precisa ser ajustada; não é automático.
